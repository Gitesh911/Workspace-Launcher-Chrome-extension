# 🚀 Workspace Launcher

A lightweight, premium, productivity-focused browser extension built for modern power users. Instantly launch curated sets of website tabs (Workspaces) matching your current flow state with native, customizable browser keyboard shortcuts. 

The user interface is meticulously styled to align with modern Chromium aesthetics, making it feel like a built-in feature rather than a third-party add-on.

![Workspace Launcher Icon](src-tauri/icons/icon.ico) ---

## ✨ Features

* **⚡ Lightning Fast Execution:** Launch complex sets of links completely instantly via background service scripts.
* **⌨️ Global Custom Hotkeys:** Use out-of-the-box system keybindings (`Ctrl+Shift+1`, `Ctrl+Shift+2`) to split your work setups without clicking an icon. Customizable via `chrome://extensions/shortcuts`.
* **🎨 Browser-Native Design:** Clean, distraction-free pop-up panel that beautifully mirrors native browser settings screens.
* **📂 Fully Dynamic Workspaces:** Easily rename setups, add links dynamically, or delete paths directly inside the interactive dashboard.
* **🛡️ Secure & Privacy-First:** Runs entirely locally inside your browser using Manifest V3 criteria and `chrome.storage.local`. No cloud tracking, accounts, or telemetry data collection.

## 🛠️ Built With
